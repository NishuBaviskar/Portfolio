# Modern Developer Portfolio 2026

A highly professional, responsive, and animated personal portfolio website built with HTML5, CSS3, and Vanilla JavaScript. Features advanced UI concepts like glassmorphism, gradient text, scroll animations, and an interactive particle background.

## Features
- **Mobile-First Responsive Design**
- **Glassmorphism UI Elements**
- **Dark & Light Mode Toggle**
- **Interactive Particle Background** (Particles.js)
- **Scroll Reveal Animations** (AOS & GSAP)
- **Typing Text Effects** (Typed.js)
- **Animated Skill Progress Bars**
- **Modern Project Showcase**

## Tech Stack
- HTML5 & CSS3
- Vanilla JavaScript
- GSAP (GreenSock Animation Platform)
- AOS (Animate On Scroll)
- Typed.js
- Particles.js
- Font Awesome

## Folder Structure
Follows the standard `index.html` with separate directories for `css/`, `js/`, and `assets/`."# Portfolio"
